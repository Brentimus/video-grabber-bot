# Video Grabber Bot

Telegram-бот, который автоматически скачивает видео из YouTube, TikTok и Instagram. Отправьте ссылку в чат — бот скачает видео и пришлёт его как видеосообщение. Поддерживает inline-режим и работу в группах через @mention.

## Поддерживаемые платформы

**YouTube**
- Обычные ссылки (`youtube.com/watch?v=...`)
- Короткие ссылки (`youtu.be/...`)
- Shorts (`youtube.com/shorts/...`)

**TikTok**
- Полные ссылки (`tiktok.com/@user/video/...`)
- Короткие ссылки (`vm.tiktok.com/...`, `vt.tiktok.com/...`, `tiktok.com/t/...`)

**Instagram** (требуются cookies)
- Reels (`instagram.com/reel/...`)
- Посты с видео (`instagram.com/p/...`)
- IGTV (`instagram.com/tv/...`)

## Быстрый старт

1. Клонируйте репозиторий:

```bash
git clone <repo-url>
cd video-grabber-bot
```

2. Создайте файл `.env`:

```bash
cp .env.example .env
```

3. Заполните `.env` своим токеном и ID администратора:

```
BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz
ADMIN_ID=123456789
```

4. (Опционально) Для Instagram — добавьте `cookies.txt`:
   - Установите расширение **"Get cookies.txt LOCALLY"** в Chrome
   - Залогиньтесь в Instagram
   - Экспортируйте куки и сохраните как `cookies.txt` в корень проекта

5. Запустите:

```bash
docker compose up -d
```

## Переменные окружения

| Переменная   | Обязательная | По умолчанию | Описание                                  |
|-------------|-------------|-------------|-------------------------------------------|
| `BOT_TOKEN` | да          | —           | Токен Telegram-бота от BotFather          |
| `ADMIN_ID`  | да          | —           | Telegram ID администратора бота           |
| `MAX_HEIGHT`| нет         | `720`       | Максимальная высота видео (px)            |

## Запуск без Docker

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
export BOT_TOKEN=your-token
python bot.py
```

Требуется установленный `ffmpeg` в системе.

## Inline-режим

Бот поддерживает inline-режим — введите `@имя_бота <ссылка>` в любом чате, чтобы увидеть превью видео и отправить ссылку. Если бот добавлен в этот чат, он автоматически скачает видео.

Для включения inline-режима: откройте @BotFather → `/setinline` → задайте placeholder (например, `Вставьте ссылку на видео`).

## @mention в группах

В групповых чатах упомяните бота через `@имя_бота` вместе со ссылкой — бот скачает видео. Работает с включённым privacy mode (по умолчанию). Для реакции на все ссылки без @mention — отключите privacy mode через `/setprivacy` в @BotFather или сделайте бота админом.

## Белый список

Управление доступом к боту в личке и inline-режиме (в группах ограничений нет). Команды доступны только администратору (`ADMIN_ID`):

| Команда                  | Описание                                     |
|--------------------------|----------------------------------------------|
| `/allow @username`       | Добавить пользователя по юзернейму           |
| `/allow 123456789`       | Добавить пользователя по ID                  |
| `/allow` (реплай)        | Добавить автора сообщения, на которое реплай  |
| `/deny @username`        | Удалить пользователя                         |
| `/whitelist`             | Показать текущий белый список                |

Пока белый список пуст — бот доступен всем. Как только добавлен хотя бы один пользователь — доступ ограничен.

## Особенности

- Видео больше 50 МБ автоматически сжимается через ffmpeg
- Защита от переполнения диска (лимит скачивания 500 МБ)
- Вертикальные видео скачиваются в нормальном качестве (до 720p)
- Временные файлы удаляются сразу после отправки

## Ограничения

- Максимальный размер файла для Telegram — 50 МБ (лимит Bot API)
- Куки Instagram могут протухнуть — потребуется повторный экспорт
